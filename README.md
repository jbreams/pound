# Pound

Pound is a minimal terminal-based text editor.

### Features
* Regex-based find with highlighting
* No dependencies other than a C++14 runtime
* Undo
* Multiple file buffers

## Building

```
$ git clone https://github.com/jbreams/pound.git
$ mkdir -p pound/build
$ cd pound/build
$ cmake ..
$ make
$ ./pound
```
